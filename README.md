# pong-rl 🏓
 RL for Pong using pytorch

## Built With 🛠
- Pygame for building the base game
- Pytorch for the RL agent and modules

## How to train 🏃‍♀️
```
poetry shell
poetry install
$env:PYTHONPATH=pwd
python src/train_rl.py
```
